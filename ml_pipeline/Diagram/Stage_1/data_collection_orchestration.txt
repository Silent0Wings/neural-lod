@startuml Stage_1_DataCollection_Orchestration

title Stage 1: Data Collection Orchestration (Activity - Lossless Wide)

skinparam conditionStyle diamond
skinparam defaultTextAlignment left

|#AliceBlue|Initialization & Run Setup|
start
:DataCollectionOrchestrator
Awake();
note right
  Initialize references:
  - CameraPathAnimator
  - MetricLogger
  - LODBiasController
  - RunTerminator
end note

:BuildRunList()
Generate all (bias, speed, rotation) tuples;
note right
  biasValues = [0.1, 0.5, 1.0, 2.0]
  moveSpeeds = [3.0, 5.0, 8.0]
  rotateSpeeds = [3.0, 5.0]
  Creates cartesian product
end note

:_totalRuns = biases × speeds × rotations;

|#GhostWhite|Warmup & Control|
repeat
  :lodController.UpdateBias(bias)
  → Set QualitySettings.lodBias;

  :cpa.ResetPath(newMoveSpeed)
  → Reset camera to start;

  :cpa.moveSpeed = speed
  cpa.rotateSpeed = rotation;

  :cpa.IsPaused = false
  → Camera starts moving;

  :logger.enabled = false
  → Prepare for warmup;

  :_waitingForWarmup = true
  _warmupTimer = 0;
  note right
    Warmup delay = 1.5s
    Allows GPU/CPU stabilization
    Camera is moving during warmup
  end note

  :Wait for warmupDelay seconds;

|#AliceBlue|Metric Logging Loop|
  :logger.ResetLogger(index, speed, rot)
  → Open new CSV file
  → Reset _validRows counter;

  :logger.enabled = true
  → Start collecting metrics
  (24-column CSV, 5000 row target);

  :progression = _currentRun / _totalRuns;

  repeat
    :Frame Loop:
    MetricLogger.Update()
    - Capture profiler data
    - Log row to buffer
    - Check row cap (5000 rows);
  repeat while (row cap reached?) is (no)
  ->yes;

|#GhostWhite|Completion & Output|
  :MetricLogger.StopLogging()
  → Flush remaining rows to CSV;

  :RunTerminator.OnRunComplete()
  → Reset scene state;

  :Write output CSV
  → ml_pipeline/data/
  training_data_bias_X_spd_Y_rot_Z.csv;

repeat while (more runs exist?) is (yes)
->no;

:All runs completed;
:_allDone = true;

note right
  **Output Structure:**
  Columns: frame_idx, cpu_frame_time_ms, gpu_frame_time_ms,
  triangles, draw_calls, move_speed, rotate_speed,
  lod_bias_current, camera_position_xyz, camera_rotation_xyz,
  screen_coverage, angular_velocity, frame_headroom_ms
end note

stop

@enduml

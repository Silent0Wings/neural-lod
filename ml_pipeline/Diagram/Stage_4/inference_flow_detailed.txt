' --- 1. DEFINITIONS (ALL FIELDS RESTORED) ---

component "**GameState**\n13 raw features\ncpu/gpu frame time\nfps, visible renderer count\ntriangle count, draw call count\ncamera speed" as GameState

component "**Extraction**\nRLFeatureExtractor\nExtract 13 features\nLoad scaler JSON\nNormalize: (f-m)/s\nFixed feature order" as Extraction

component "**Normalized**\n13-dim vector\nMean = 0, Std = 1\nReady for MLP" as Normalized

component "**MLP**\nL1: 13→256 ReLU\nL2: 256→64 ReLU\nL3: 64→32 ReLU\nOut: 32→1 Linear\n~21,857 params\nInference: 0.5-1ms" as MLP

component "**RawAction**\nUnbounded output\nmu = tanh(net)*0.2\nRange: [-0.2, +0.2]\nGaussian policy" as RawAction

component "**DeadZone**\nif abs(mu) < 0.02 → 0\nSuppress GPU noise\nFilter 1 of 4" as DeadZone

component "**Smoothed**\nmu = 0.2*raw + 0.8*prev\nEMA alpha = 0.2\nDampen oscillations" as Smoothed

component "**DwellTimer**\nif frames < 5 → action = 0\nEnforce 0.5s minimum\nPrevent rapid switch" as DwellTimer

component "**Clamped**\naction in [-0.2, +0.2]\nbias in [0.30, 2.00]\nRuntime guardrails\nRecovery assist" as Clamped

component "**Applied**\nUpdate QualitySettings\nApply lodBias delta\nCurrent frame active\nNext frame: measure" as Applied

component "**Logged**\nLog CSV row\naction_raw, action_final\ncpu_ms, gpu_ms\nreward components" as Logged

' --- 2. SNAKE LAYOUT LOGIC ---

' Row 1: Right
GameState -r-> Extraction
Extraction -r-> Normalized
Normalized -r-> MLP

' Turn 1: Down
MLP -d-> RawAction

' Row 2: Left
RawAction -l-> DeadZone
DeadZone -l-> Smoothed
Smoothed -l-> DwellTimer

' Turn 2: Down
DwellTimer -d-> Clamped

' Row 3: Right
Clamped -r-> Applied
Applied -r-> Logged

note bottom of Logged
  **TRAINING FEEDBACK:**
  Collected rollouts flow to training
  Compute discounted returns
  REINFORCE: maximize E[log(pi)*G]
  Update network weights via gradient
end note

@enduml

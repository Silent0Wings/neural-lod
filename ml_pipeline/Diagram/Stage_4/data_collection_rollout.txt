@startuml Stage_4_DataCollection_Rollout

title Stage 4: RL Rollout Collection (Activity)

start

:RLRolloutLogger.Start()
→ RLFeatureExtractor ready;

note right
  RLFeatureExtractor loads:
  - Scaler JSON (13-feature normalization)
  - Runtime contract constants
  - Recovery/correction eligibility thresholds
end note

repeat
  :RLRolloutLogger.StartEpisode()
  → Create new CSV: rollout_ep{N}_{timestamp}.csv;

  note right
    Episode parameters:
    - _episodeIndex = N
    - _stepIndex = 0
    - _loggingActive = true
  end note

  :QualitySettings.lodBias = 1.0f
  (Reset to neutral);

  :_extractor.ResetEpisodeState()
  → Clear dwell scores, switch ring buffer;

  :_policyController.ResetEpisode()
  → Reset bias + dwell counter;

  repeat
    :RLPolicyController.Update()
    (inference interval = every 2 frames);

    :RLFeatureExtractor.Update()
    → Extract 13 features
    (cpu, gpu, fps, renderers, triangles,
     draw_calls, speed, rot_speed, coverage,
     previous_bias, lod_switches, dwell_scores);

    :Normalize 13 features
    using loaded scaler (mean/scale);

    :Run inference on RL policy ONNX
    → Get raw action delta
    (unbounded from linear output);

    :RLPolicyController.ApplyWithGuardrails()
    → Snapshot bias before action
    → Apply: dead zone, EMA, dwell, recovery;

    :Set QualitySettings.lodBias
    (committed in next frame);

    if (ShouldLogDecisionFrame?) then (yes)
      :RLRolloutLogger.LateUpdate()
      → Write CSV row (23 columns);

      note right
        Cols 1-15: 13 features
        Cols 16-23: bias states + actions
      end note

      :_stepIndex++;
      :_rowBuffer.Add(row);

      if (_rowBuffer.Count >= 120) then (yes)
        :Flush rows to disk;
      else (no)
      endif

    else (no)
    endif

  repeat while (stepsPerEpisode < _stepIndex?)

  :RLRolloutLogger.EndEpisode();

  :Flush remaining rows;
  :Close file;

  note right
    Episode output:
    - rollout_ep{N}_{timestamp}.csv
    - {stepsPerEpisode} rows logged
    - Path: Application.persistentDataPath/RLRollouts/
  end note

  :_episodeIndex++
  :_extractor.ResetEpisodeState();

repeat while (maxEpisodes < _episodeIndex?)

:CollectionComplete = true;

:Python training reads all CSVs from RLRollouts/
→ Apply df_samples_clean filtering
→ Compute discounted returns (gamma=0.99)
→ REINFORCE: gradient update on policy;

note right
  Training signal from GPU frame time only
  Python reward formula:
  r_t = (gpu_prev - gpu_current)
        + BONUS * (gpu_current <= target_ms)
  Clipped to [-5.0, +5.0]
end note

stop

@enduml
